### Project Overview
This project aims to develop a Teaching Assistant (TA) recruitment management system for the BUPT International School using Agile methodologies, streamlining the traditional paper-and-Excel workflow by implementing core functions such as job posting, CV submission, application tracking, and AI-powered candidate matching to enhance recruitment efficiency and workload balancing.
